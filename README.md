# SeedDK
SeedDK is a repository for research, reverse engineering, tooling and editors for Sword Art Online:  Hollow Fragment.
The name is clever too.

## Data Crystal ##
https://datacrystal.tcrf.net/wiki/Sword_Art_Online:_Hollow_Fragment
